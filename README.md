# FilePublication-Angular-Node-Mysql

### FilePublication with technology stack NodeJs, Angular, Mysql, JWT, Javascript

## Database
Database dump in database_schema directory

## Backend - NodeJs, Mysql, JWT
Install all package   
`npm i`  
Run  
`nodemon server.js`

## Frontend - Angular

`cd client`

Install all package  
`npm i`  
Run  
`npm start`
